# crop-detection
**Steps to run the application**

**Step 1 :** First run the .ipynb file from google colab. After running all the cells, these files will be generated: crop_label_encoder.pkl, crop_scaler.pkl, final_crop_recommendation_model.pkl. You can also download the file from github for your convinence.

**Step 2 :** After downloading the file and templates folder run the following command on bash or command prompt. 

**cd /d file_location
python app.py**

now localhost link will be genated in the command prompt. Copy and paste that link in your preferable browser.

**File Description: **
1. The three **.pkl** files are the model files.
2. **app.py** is the application file through which the model will run.
3. **index.html** is the application front-end file.
4. **Crop_recommendation.csv** is the dataset

**N.B. The index.html file must need to be inside templates folder.**
